# src/ml/ — ONNX Model Files

This folder contains the compiled ONNX models and the feature order JSON used
by `mlInference.service.ts` at runtime.

## Files

| File | Description |
|------|-------------|
| `purchase_model.onnx` | XGBClassifier — outputs purchase probability [0, 1] |
| `value_model.onnx`    | XGBRegressor — outputs expected order value (float) |
| `feature_order.json`  | Exact column order used during training. Node reads this to build input tensors |

## ⚠️ Regeneration

These files are generated by the Python training script in `../../ml/train_models.py`.

**If the CustomerProfile or Conversation schema changes** (new features, renamed fields,
changed ranges), you MUST re-run the training script and re-copy the outputs here:

```bash
cd ../../ml
pip install -r requirements.txt
python train_models.py
# Then copy the outputs:
cp models/purchase_model.onnx ../omniflow-backend/src/ml/
cp models/value_model.onnx    ../omniflow-backend/src/ml/
cp models/feature_order.json  ../omniflow-backend/src/ml/
```

Do NOT hand-edit `feature_order.json` — the JSON and the ONNX model weights must
always correspond to the same training run.
